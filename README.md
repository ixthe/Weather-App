# Weather App

Aplicação simples de clima que permite buscar informações meteorológicas de qualquer cidade em tempo real.

---


##  Funcionalidades

-  Busca de cidade
-  Exibição de temperatura
-  Ícones dinâmicos de clima
-  Interface moderna e responsiva

---

##  Tecnologias utilizadas

- HTML5  
- CSS3  
- JavaScript  

## Melhorias futuras

-  Alterar o fundo do app dinamicamente com base na cidade pesquisada
- Exibir ilustrações turísticas de acordo com a localização
-  Identificar país/estado automaticamente a partir da cidade

### Exemplos de ideia

- Foz do Iguaçu → Cataratas do Iguaçu  
- Rio de Janeiro → Cristo Redentor  
- Paris → Torre Eiffel  
- New York → Skyline da cidade
